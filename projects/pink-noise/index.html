Pink Noise

Pink Noise is an ongoing live coding and modular synthesis project that explores the stochastic properties of electronic sound. Using SuperCollider and a Moog Matriarch synthesizer, I generate self-evolving sonic textures that blur the line between composition and generative process.

Each performance is a unique algorithmic improvisation where code becomes instrument. The project investigates how randomness and controlled chaos can produce emotionally resonant musical structures, drawing from both electroacoustic tradition and contemporary computer music.
