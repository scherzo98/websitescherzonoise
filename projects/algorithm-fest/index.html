Live Coding Festival

{PinkNoise.ar} Algorithm Music Fest

The festival was held in 2020 as part of the International Science Fiction Festival of Bogotá, Colombia — Festival Paralelo.

Algorave / Mozilla Hubs Performance

This live coding performance took place within Mozilla Hubs, a virtual reality social platform, as part of an Algorave event organized by the collective Mujeres al Borde del Ruido, a nodal project of the live coding network La Salpicodera and TOPLAP Bogotá.

The performance explored live coding as a real-time audiovisual and musical practice in which sound and visual structures are generated through code during the performance itself. Situated within the context of Latin American live coding communities, the event investigated the intersection between algorithmic composition, improvisation, networked performance, and virtual environments.

Developed in a shared VR space, the concert proposed an immersive and collective listening experience where performers, audiences, and digital space interacted simultaneously. The project reflects an interest in live coding not only as a compositional tool, but also as a performative and social practice capable of creating collaborative and participatory sonic environments through code.

The performance formed part of the broader ecosystem of live coding initiatives developed by La Salpicodera in Bogotá since 2018, contributing to spaces of experimentation, knowledge exchange, and the dissemination of algorithmic art and sound practices in Colombia and Latin America.
